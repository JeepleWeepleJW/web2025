공개 API

웹 API

웹 스토리지

  • 웹 브라우저 저장 공간
  • 세션 스토리지, 로컬 스토리지로 구분

쿠키와 웹 스토리지

  • 웹 스토리지 API 전에는 쿠키에 정보 저장
  • 쿠키는 파일 형태로 최대 300개까지 저장 가능
  • 각 도메인당 50개까지 저장 가능
  • 웹 스토리지는 도메인당 2~10MB 저장 가능

세션 스토리지

  • 세션 동안만 데이터 유지
  • 하나의 탭은 하나의 세션

로컬 스토리지

  • 세션이 끝나도 계속 데이터 보관
