챕터8

window

  • 웹 브라우저 창을 관리하는 객체
  • 프로퍼티에 접근하려면 window.프로퍼티명

프로퍼티

프로퍼티기능 설명
document웹 브라우저 창에 표시된 웹 문서에 접근할 수 있습니다.
frameElement현재 창이 다른 요소 안에 포함되어 있으면 해당 요소를, 없으면 null 반환.
innerHeight내용 영역의 높이
innerWidth내용 영역의 너비
localStorage로컬 스토리지 접근
location현재 URL 정보
name브라우저 창의 이름
outerHeight바깥쪽 높이
outerWidth바깥쪽 너비
pageXOffset수평 스크롤 거리 (scrollX와 같음)
pageYOffset수직 스크롤 거리 (scrollY와 같음)
parent부모 프레임
screenX왼쪽 테두리로부터 거리
screenY위쪽 테두리로부터 거리
scrollX수평 스크롤 거리
scrollY수직 스크롤 거리
sessionStorage세션 스토리지 접근

window 객체의 메서드는 window 생략 가능

메서드

메서드기능 설명
alert()알림 창 표시
blur()포커스 제거
close()현재 창 닫기
confirm()확인 창 표시
focus()포커스 설정
moveBy()크기만큼 이동
moveTo()좌표로 이동
open()새 창 열기
postMessage()다른 창에 메시지 전달
print()문서 인쇄
prompt()텍스트 입력창
resizeBy()크기만큼 조절
resizeTo()지정 크기로 조절
scroll()특정 위치로 스크롤
scrollBy()크기만큼 스크롤
scrollTo()지정 위치까지 스크롤
setCursor()커서 변경
showModalDialog()모달 창 열기
sizeToContent()내용에 맞게 창 크기 조절
stop()웹 문서 로딩 중지

팝업창 열기 - open

window.open(경로, 창 이름, 창 옵션)

팝업창 닫기 - close

open으로 연 창을 닫습니다.
