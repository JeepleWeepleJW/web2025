인라인 스크립트

  • HTML 태그 안에서 직접 작성하는 방법
  • 간단한 명령을 처리할 경우 주로 사용
  • 예시 코드: 버튼을 클릭해서 알림 창 표시 
    <button onclick="alert('알림 메시지가 표시됩니다.')">클릭!</button>

내부 스크립트

  • <script> 태그와 </script> 태그를 사용해서 자바스크립트 소스를 HTML 문서 안에 작성
  • <script> 태그는 문서 어디든 위치할 수 있고, 삽입된 위치에서 바로 실행
  • 하나의 문서 안에 여러 개의 <script> 태그 사용 가능
  • 보통 </body> 태그 직전에 삽입
  • 예시 코드: 텍스트 클릭 시 배경색 변경 
    <body>
  <p>새로 고칭해 보세요</p>

  <script>
    function random(number) {
      return Math.floor(Math.random() * number);
    }

    function bgChange() {
      const rndCol = 'rgb(' + random(255) + ',' + random(255) + ',' + random(255) + ')';
      document.body.style.backgroundColor = rndCol;
    }

    bgChange();
  </script>
</body>

외부 스크립트

  • 자바스크립트 소스를 .js 파일로 따로 저장하고 HTML 문서에 링크
  • HTML 문서가 깔끔해지고, 코드 재사용 및 유지보수가 쉬움
  • 예시 코드: 
    <script src="스크립트 파일 경로"></script>
<script defer src="스크립트 파일 경로"></script>

인라인 스크립트와 내부 스크립트의 단점

  • 가독성 저하
  • 여러 문서에서 동일한 소스를 반복 작성해야 함
  • 수정 시 모든 문서에서 하나하나 수정해야 함
