전개 구문

  • 어디에 사용하느냐에 따라서 ‘나머지 매개변수’ or ‘전개 구문’ 이라고 한다.

객체의 프로퍼티

  • 점 표기법을 사용해서 접근
  • 에크마스크립트에서는 대괄호로 접근 가능

전역 심벌

  • 여러 곳에서 같이 사용할 수 있는 심벌

map() 메서드

  • 결과를 새로운 배열로 반환하는 메서드

filter() 메서드

  • 특정 조건에 맞는 요소만 골라내는 메서드

reduce() 메서드

  • 배열 원소에서 차례대로 함수를 실행해서 하나의 결과를 만드는 것

Map 객체

  • 키와 값이 하나의 쌍으로 구성
