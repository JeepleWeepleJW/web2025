웹의 3대 요소

  • HTML: 정보의 표현
    • 화면 표시, 문서 구조 표현, 정보 표시
    • 태그를 이용하여 표현
  • HTTP: HTML 형식의 정보를 교환하는 통신 규약
    • 인터넷 프로토콜 중 하나
  • URL: 인터넷 자원을 표시하는 통일된 방식

HTML 문서 구조

  • Head: 브라우저에게 필요한 정보 제공 (화면에 표시되지 않음)
  • Body: 브라우저 화면에 표시되는 부분

Hyper

  • Hyper Link:
    • 주소와 주소가 비선형적으로 엮인 인터페이스
    • 하이퍼텍스트 문서 안에서 모든 형식의 자료를 직접 연결할 수 있는 참조 고리
  • Hyper Text:
    • 참조를 통해 한 문서에서 다른 문서로 즉시 접근 가능한 텍스트
    • 문장이나 단어 등이 링크로 연결된 네트워크 구조의 문서
