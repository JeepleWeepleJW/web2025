자바스크립트 입·출력 및 변수·자료형 기초

1. alert()

알림 창을 표시하는 데 사용합니다.

alert(내용);
alert("안녕하세요?");
let a = 10;
alert(a);

2. confirm()

확인(OK/취소) 창을 표시하고, 사용자의 응답을 불리언으로 반환합니다.

confirm(내용);
confirm("프로그램을 종료하겠습니까?");

if (confirm("제출하시겠습니까?")) {
  alert("정상적으로 제출되었습니다.");
} else {
  alert("제출실패");
}

3. prompt()

사용자가 간단한 값을 입력할 수 있는 창을 표시합니다. 취소하면 null을 반환합니다.

prompt(내용);
prompt("이름을 입력하세요");

prompt(내용, 기본값);
prompt("컨퍼런스 참석 여부(예/아니요)", "예");

4. console.log()

개발자 도구 콘솔 창에 텍스트나 변수 값을 출력할 때 사용합니다.

console.log(내용);
console.log(변수);
console.log("자바스크립트를 공부 중입니다.");

let name = "도레미";
console.log(name + "님은 자바스크립트를 공부 중입니다.");

5. document.write()

웹 브라우저 창에 직접 내용을 표시합니다.

let name = "도레미";
document.write("제 이름은 " + name + "입니다.");

6. 변수(Variable)

값을 담아 놓는 바구니 같은 존재입니다.

6.1. 변수 이름 규칙

  • 공백 사용 불가
  • 대소문자 구분
  • 한 단어 변수: 영어 소문자 사용
    두 단어 이상: 언더스코어(_)나 카멜케이스 사용
    예: total_area, totalArea
  • 예약어(let, const 등) 사용 불가
  • 의미 없는 이름 피하기

6.2. 변수 선언과 값 할당

변수 이름 지정 → 변수 선언
=를 사용해 값 할당

// 선언만
let x;

// 선언과 동시에 할당
let y = 5;

// 상수 선언 (재할당 불가능)
const PI = 3.14;

const는 선언 시 반드시 초기값을 지정해야 하며, 재할당 시 오류 발생

6.3. var와 호이스팅

var로 선언한 변수는 호이스팅(끌어올리기)되어 선언 전에 접근해도 undefined 상태로 존재합니다.

var x = 10;
var sum = x + y;
var y = 20;
console.log(sum);  // NaN

let이나 const는 호이스팅 시에도 선언 전 사용하면 오류 발생

6.4. 변수 재선언 & 재할당

  • var: 재선언 및 재할당 가능
  • let: 재선언 불가, 재할당 가능
  • const: 재선언 불가, 재할당 불가

7. 자료형(Data Type)

값의 종류(숫자, 문자열 등)를 나타내며, 크게 “원시 유형(Primitive)”과 “객체(Object)”로 나뉩니다.

7.1. 원시 유형

  • 숫자(number)
  • 문자열(string)
  • 논리형(boolean)
  • undefined
  • null
  • symbol

여러 원시 값을 저장하려면 배열, 여러 속성을 묶으려면 객체를 사용합니다.

7.2. typeof()

값이나 변수의 자료형을 문자열로 반환합니다.

let data = 5;
typeof data;    // "number"
typeof "hello"; // "string"
typeof true;    // "boolean"

7.3. 숫자형(Number)

정수와 실수 모두 포함. 따옴표로 묶으면 문자열로 취급됩니다.

typeof 10;    // "number"
typeof "10";  // "string"
typeof 3.14;  // "number"

7.4. 문자열(String)

작은따옴표 또는 큰따옴표로 묶습니다. 내부에 같은 따옴표 사용 시 이스케이프를 활용하거나 다른 따옴표로 감싸세요.

typeof "안녕하세요?"; // "string"
console.log('I\'m studying now.');
console.log("탭 \t 포함");

7.5. 템플릿 리터럴(Template Literal)

백틱(`)을 사용해 변수·식 삽입 가능

let name = "백두산";
let room = 205;
console.log(`${name}님, ${room}호 강의실로 입장하세요.`);

7.6. 논리형(Boolean)

true 또는 false 만 사용합니다.

7.7. Truthy & Falsy

불리언 컨텍스트에서 암묵적으로 true 또는 false 로 평가되는 값들

TruthyFalsy
Falsy 이외의 모든 값0, "", NaN, undefined, null

7.8. undefined & null

undefined: 값이 할당되지 않은 변수의 초기 상태
null: 의도적으로 “값이 없음”을 나타내는 값

7.9. 객체(Object)

키-값 쌍으로 구성된 복합 자료형. 배열, 함수도 객체로 취급됩니다.

let book = {
  title: "JavaScript Guide",
  author: "MDN"
};
console.log(book.title);    // "JavaScript Guide"
console.log(book["author"]); // "MDN"

7.10. 배열(Array)

여러 값을 순서대로 저장하는 자료형. 인덱스는 0부터 시작.

let colors = ["red", "blue", "green"];
console.log(colors.length); // 3
console.log(colors[1]);     // "blue"

7.11. 심벌(Symbol)

유일성이 보장되는 원시 값으로, 객체 속성 키로 자주 사용됩니다.

let id1 = Symbol("id");
let id2 = Symbol("id");
console.log(id1 === id2); // false

let obj = {};
obj[id1] = 123;
console.log(obj[id1]); // 123

7.12. 느슨한 vs. 강력한 자료형 체크

자바스크립트는 느슨한(동적) 타입 언어로, 값에 따라 자료형이 결정됩니다. 협업 시 일관된 변수 사용을 위해 타입스크립트 같은 강력한 타입 검사 도구를 활용하기도 합니다.

7.13. 자동 형 변환(Type Coercion)

문자열과 숫자를 사칙연산할 때 자바스크립트가 자동으로 타입 변환을 수행합니다.

let one = "20";
let two = 10;

console.log(one + two); // "2010" (문자열 연결)
console.log(one - two); // 10     (숫자로 변환 후 계산)

7.14. Number() 함수

값을 숫자로 변환합니다.

Number(true);  // 1
Number("20");    // 20
Number("Hi?");   // NaN

7.15. parseInt() & parseFloat()

parseInt()는 정수로, parseFloat()는 실수로 변환합니다.

7.16. toString() & String()

값을 문자열로 변환합니다.

let num = 10;
console.log(num.toString());    // "10"
console.log(num.toString(2));   // "1010" (2진수 문자열)

let flag = false;
console.log(String(flag));      // "false"
console.log(String(null));      // "null"

7.17. Boolean() 함수

값을 불리언으로 변환합니다. 0, 빈 문자열, null, undefined, NaN는 false로 평가됩니다.

Boolean(1);      // true
Boolean(0);      // false
Boolean("text"); // true
Boolean("");     // false
