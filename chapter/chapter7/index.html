Chapter7

  • 모든 HTML 태그는 요소(element) 노드가 됩니다.
  • HTML 태그에서 사용하는 텍스트 내용은 자식 노드인 텍스트(text) 노드가 됩니다.
  • HTML 태그에 있는 속성은 모두 자식 노드인 속성(attribute) 노드가 됩니다.
  • 주석은 주석(comment) 노드가 됩니다.

노드리스트

querySelectorAll()로 가져올 수 있습니다.

노드 생성

document.createElement("p")

텍스트 노드 생성

document.createTextNode("TypeScript")

자식 노드 연결

부모노드.appendChild(요소)

속성 노드 만들기

const attr = document.createAttribute("속성명");
attr.value = "속성값";

속성 노드 연결하기

노드.setAttributeNode(attr);

노드 삭제하기

요소.remove();

자식 노드 삭제하기

부모노드.removeChild(자식노드);
