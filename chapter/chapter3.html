산술 연산자

+, -, *, /, %, ++, -- 있다.

비교 연산자

연산자 기능 사용 예
== 피연산자 값이 같으면 true 3 == "3" → true
=== 피연산자 값과 데이터 유형이 모두 같으면 true a === "3" → false
!= 피연산자 값이 같지 않으면 true 3 != "3" → false
!== 피연산자 값과 데이터 유형이 모두 같지 않으면 true 3 !== "3" → true
< 왼쪽 값이 오른쪽 값보다 작으면 true 3 < 4 → true
<= 왼쪽 값이 오른쪽 값보다 작거나 같으면 true 3 <= 4 → true
> 왼쪽 값이 오른쪽 값보다 크면 true 3 > 4 → false
>= 왼쪽 값이 오른쪽 값보다 크거나 같으면 true 3 >= 4 → false

Switch 예시

기본형
switch (조건) {
    case 값:
        문장;
        break;
    case 값:
        문장;
        break;
    …
    default:
        문장;
}

For문

자바스크립트에서 가장 많이 사용하는 반복문

// for (초기값; 조건; 증가식) { … }
const students = ["Park", "Kim", "Lee", "Kang"];

for (let i = 0; i < students.length; i++) {
  document.write(`${i + 1}번 학생 이름은 ${students[i]}입니다.
`);
}

forEach문

배열에서 forEach를 사용해 좀 더 쉽게 반복

const students = ["Park", "Kim", "Lee", "Kang"];

students.forEach(function(student) {
    document.write(`${student}.`);
});

for…in 문

객체나 배열(객체 취급)에 대해 키를 반복해서 처리 가능

const gitBook = {
    title : "깃&깃허브 입문",
    pubDate : "2019-12-06",
    pages : 272,
    finished : true
};

for (key in gitBook) {
    document.write(`${key} : ${gitBook[key]}
`);
}

for…of 문

문자열이나 배열 등 반복 가능한 자료에 사용

const students = ["Park", "Kim", "Lee", "Kang"];

// students에 student가 있는 동안 계속 반복합니다.
for (student of students) {
    document.write(`${student}. `);
}

while

조건이 true인 동안 반복

while (조건) {
    실행할 명령
}

do…while

최소 한 번 실행 후 조건을 체크

do {
    실행할 명령
} while (조건);

continue
